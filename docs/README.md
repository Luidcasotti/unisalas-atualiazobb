# Documentacao do UniSalas

Esta pasta concentra os materiais de apoio do projeto.

## Arquivos principais

- `DOCUMENTACAO_ESSENCIAL_UNISALAS.md`: resumo direto do sistema, tecnologias e funcionamento.
- `DOCUMENTACAO_ESSENCIAL_UNISALAS.pdf`: versao em PDF da documentacao essencial.
- `DOCUMENTACAO_COMPLETA_UNISALAS.md`: documentacao mais detalhada do projeto.
- `DOCUMENTACAO_COMPLETA_UNISALAS.pdf`: versao em PDF da documentacao completa.
- `database_unisalas.sql`: script consolidado para criar/importar o banco com dados iniciais.

## Materiais de TCC

- `tcc_luid_corrigido.pdf`
- `tcc_luid_atualizado_mudancas.pdf`
- `tcc_luid_mudancas_anexo.pdf`

## Pastas de renderizacao

As pastas `render_check*` guardam imagens usadas para conferencia visual dos PDFs gerados.
